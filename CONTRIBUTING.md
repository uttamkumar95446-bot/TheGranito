## How to Contribute

- **Fork the repo** on GitHub (https://github.com/uttamkumar95446-bot/TheGranito).
- **Clone your fork** locally: `git clone https://github.com/YOUR_USERNAME/TheGranito.git`.
- **Create a branch** for your changes: `git checkout -b feature/your-feature-name`.
- Make your changes, focusing on clean code that matches existing styles (e.g., semantic HTML, responsive CSS).[web:3][web:4]
- **Test locally** by opening `index.html` in a browser.
- **Commit changes** with clear messages: e.g., "Add responsive navbar" or "Fix footer alignment".
- **Push to your fork** and open a Pull Request (PR) to the main repo.[web:4][web:10]

## What We're Looking For

- Improvements to portfolio sections (e.g., projects, about, contact).
- Bug fixes for responsiveness or cross-browser issues.
- Documentation updates or examples for others to replicate this setup.
- New features like dark mode or animations (discuss in an issue first).[web:3][web:5]

## Coding Style

- Use **semantic HTML5**.
- **CSS**: Flexbox/Grid preferred; keep it vanilla or note any frameworks.
- **JS**: Vanilla JS; ES6+ syntax.
- Indent with 2 spaces; no trailing whitespace.[web:3][web:4]

## Reporting Bugs or Suggesting Features

- Open a new issue on GitHub.
- Include steps to reproduce, browser details, and screenshots if possible.[web:3]

## Code of Conduct

Be respectful and helpful. Contributions should benefit the project and its users.[web:5][web:8]
